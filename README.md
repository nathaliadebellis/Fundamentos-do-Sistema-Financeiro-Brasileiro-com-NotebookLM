# Fundamentos-do-Sistema-Financeiro-Brasileiro-com-NotebookLM
Desenvolvi um caderno temático no NotebookLM sobre os fundamentos do Sistema Financeiro Brasileiro. O projeto consistiu na curadoria de materiais confiáveis, definição de objetivos de aprendizagem, elaboração de perguntas estratégicas e experimentação de diferentes técnicas de prompt para explorar os conteúdos de forma crítica.
