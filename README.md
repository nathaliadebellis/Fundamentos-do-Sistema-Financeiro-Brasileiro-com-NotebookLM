## 📖 Sobre o Projeto

Este projeto foi desenvolvido como parte do **Bootcamp Bradesco – GenAI, Dados & Cyber**, com o objetivo de explorar o uso da Inteligência Artificial Generativa como ferramenta de aprendizagem ativa.

O trabalho consiste na criação de um caderno temático no **NotebookLM**, utilizando fontes oficiais sobre o Sistema Financeiro Brasileiro para construir uma base de conhecimento, realizar pesquisas, sintetizar informações e organizar conteúdos de forma estruturada.

Durante o desenvolvimento foram aplicadas técnicas de curadoria de fontes, engenharia de prompts, pensamento crítico e validação das respostas por meio das referências fornecidas pela própria ferramenta.

---

# 🎯 Objetivos

- Compreender o funcionamento do Sistema Financeiro Nacional.
- Identificar o papel do Banco Central, da CVM e das instituições financeiras.
- Entender conceitos fundamentais como inflação, taxa Selic, crédito e investimentos.
- Utilizar IA Generativa para organizar conhecimento de forma eficiente.
- Desenvolver pensamento crítico na análise das respostas geradas pela IA.

# 📖 Glossário

| Conceito | Definição |
|----------|-----------|
| Sistema Financeiro Nacional | Conjunto de instituições responsáveis pelo funcionamento do mercado financeiro brasileiro. |
| Banco Central | Responsável pela política monetária, estabilidade da moeda e fiscalização do sistema financeiro. |
| Selic | Taxa básica de juros da economia brasileira. |
| Inflação | Aumento contínuo dos preços de bens e serviços. |
| IPCA | Índice oficial utilizado para medir a inflação no Brasil. |
| Crédito | Recursos disponibilizados mediante compromisso de pagamento futuro. |
| Spread Bancário | Diferença entre o custo de captação dos bancos e a taxa cobrada dos clientes. |
| Open Finance | Compartilhamento padronizado de dados financeiros mediante autorização do cliente. |
| PIX | Sistema de pagamentos instantâneos desenvolvido pelo Banco Central. |
| Liquidez | Facilidade de converter um ativo em dinheiro. |
| Diversificação | Estratégia de distribuição dos investimentos para reduzir riscos. |

# 📌 Principais Aprendizados

- A qualidade das respostas depende diretamente da qualidade das fontes utilizadas.
- Pequenas alterações nos prompts produzem respostas significativamente diferentes.
- O NotebookLM facilita a organização e a síntese de grandes volumes de informação.
- A IA deve ser utilizada como apoio ao aprendizado e não como substituta da análise crítica.
- O uso de fontes oficiais aumenta a confiabilidade das informações produzidas.
